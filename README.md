# 📖 واجهة "عشر سنوات من التدبر" التفاعلية

هذا المشروع هو منصة عرض بيانات (Data Visualization) وتوثيق لمسيرة **مجلة تدبر القرآنية** الممتدة لعشر سنوات، بدءاً من المحرم 1438هـ وحتى رجب 1447هـ. تم تصميم الصفحة لتكون لوحة شرف رقمية تستعرض الإنجازات العلمية والعددية للمجلة بأسلوب تفاعلي عصري.

## 🧠 شرح تقني عميق (Technical Deep Dive)

تم بناء المشروع بفلسفة **"الأداء أولاً"**، حيث يعتمد كلياً على التقنيات القياسية للمتصفح دون الحاجة لمكتبات خارجية ثقيلة:

### 1\. نظام الرسوم المتحركة (Animation Engine)

  * **Intersection Observer API**: يستخدم المشروع هذه التقنية لمراقبة تمرير المستخدم، حيث لا تبدأ العدادات والرسوم البيانية في العمل إلا عند وصولها إلى منطقة الرؤية.
  * **CSS Custom Properties**: يتم التحكم في هوية الألوان (الذهبي، البني، الكريمي) عبر متغيرات مركزية في `:root` لتسهيل التخصيص.

### 2\. تمثيل البيانات (Data Representation)

  * **الرسوم البيانية**: صُممت الرسوم الدائرية (Donut Charts) باستخدام وسم `svg` والتحكم في خاصية `stroke-dasharray` برمجياً لتحديد النسب.
  * **مخطط الأعمدة**: يعتمد على Flexbox الديناميكي، حيث يتم حقن البيانات من مصفوفة JavaScript لتوليد الأعمدة وتعديل أطوالها تلقائياً.

## 🛠️ دليل التعديل وتخصيص البيانات (Customization Guide)

يمكنك تعديل محتوى الصفحة بسهولة من خلال فتح ملف `tadabbur-interactive.html`:

### أ- تعديل الأرقام والعدادات:

ابحث عن أي وسم يحتوي على الكلاس `counting` وقم بتغيير قيمة `data-target`:

```html
<span class="fc-num counting" data-target="20">0</span> 
```

### ب- تعديل بيانات المخطط السنوي:

توجه إلى جزء JavaScript وابحث عن المصفوفة `barData` لتعديل السنوات أو القيم:

```javascript
const barData = [
  { year: '١٤٤٧هـ', label: '2025-26', val: 2 },
];
```

### ج- تعديل نسب المحتوى:

لتغيير أشرطة التقدم (Progress Bars)، قم بتعديل قيمة `data-width` في القسم المخصص:

```html
<div class="prog-fill" data-width="90"></div>
```
